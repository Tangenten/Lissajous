# 2D Lissajous Curve Graphics, Can Draw Custom Curves Too

Controls <br />
k/w = lfo1 frequency <br />
a/s = lfo2 frequency <br />
e/r = lfo1 phase <br />
d/f = lfo2 phase <br />

v = update from drawn wavetable <br />
z = smooth drawn wavetable <br />
b = set wavetable to sin curve <br />
